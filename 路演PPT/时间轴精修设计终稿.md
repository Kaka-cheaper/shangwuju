# 时间轴精修设计终稿

> 目标：去掉时间轴的塑料感（彩虹脊柱/发光饱和点/糖豆chip），让脊柱和节点卡成为**一个联动的整体**，有润滑感、克制。
> 状态：三决策已锁（暖灰脊柱 / 通勤+时间挪脊柱上 / 联动全加）。与节点卡精修、信任带同族。待落地（排在信任带批提交后，与节点卡同文件顺序做）。

---

## 一、克制：把塑料感摁下去

| 元素 | 现状（塑料） | 改后（克制） |
|---|---|---|
| **脊柱线** | 彩虹渐变（绿→黄→红 / 蓝→紫→粉） | **单色暖灰** `#D8D2C4`（或同一色相极淡上→下渐变给方向感），宽 1.5px |
| **圆点** | 大号高饱和发光球 | 小（9px）、**扁平无光泽**、单色相；活动/餐厅节点=暖灰实心点，当前节点=一圈极淡 amber ring；首尾"家"=小 home 图标点（中性灰，不用 🚀/✨） |
| **时间标签** | 黑体贴脊柱 | 13px `tabular-nums` ink-500，贴脊柱、和点垂直对齐 |
| **节点标签/chip** | 满屏彩色丸 | 已由节点卡精修降饱和（本批不重复） |

美团黄只在"当前节点 ring"点一下，其余脊柱全程暖灰——这是"克制"的关键。

---

## 二、通勤 + 时间挪到脊柱上（一箭三雕）

现状：通勤是**一张独立卡**（"通勤 27 分钟(估算)"占一行），时间又**同时**在脊柱和卡里出现两处（节点卡精修引入的重复）。

改：**通勤和时间都归到脊柱上**——
- **时间**：只在脊柱上（点的上方=start、下方=end），**节点卡里的时间去掉**（消除"出现两次"）；
- **通勤**：做成脊柱两点**之间**的一小段淡标——脊柱在这段变虚线/更细 + 旁边极小灰字"🚶 27 分钟"，**不再是一张卡**。

三个收益：①去重复时间 ②减若干张通勤卡（方案区更干净）③把"行程流动"和脊柱绑成一体。

```
脊柱形态：
19:30 ●━━━━━━  ← 实线段 = 停留（节点卡在右）
        ┊
        ┊ 🚶 27 分钟   ← 虚线细段 = 通勤（无卡，脊柱旁小灰字）
        ┊
21:00 ●━━━━━━
```

## 三、联动 + 润滑感（脊柱 ↔ 卡片，静态+动态全加）

三层，由淡到强：

1. **对齐（静态）**：圆点垂直对齐节点卡的**标题行**（不是卡顶），视线从点→店名一条直线滑过；
2. **连接（静态）**：点到卡片左缘一条极短极淡的连接线（或卡片左缘一个对齐点的小缺口/tick），把"这个点属于这张卡"钉死；
3. **悬停协同（动态·真正的润滑感）**：
   - 悬停某张卡 → 它的点微放大 + 相邻脊柱段染一点暖(amber/8%) + 连接线提亮 + 卡片轻抬（translateY -1px / shadow 微增）；
   - 悬停某个点/时间 → 对应卡片轻抬高亮；
   - 双向、200ms ease、只动 transform/opacity/color。**这一条把"两摊"变"一体"。**

```
时间     脊柱  点   连接         卡片
19:30    │   ●━━━━┐  ┌────────────────────────┐
              ┊   └─→│ ⟨用餐⟩ 炭烤大叔·路边烧烤  │ ← 点对齐标题行
              ┊🚶27  │ ⇄老王烧烤摊 ⇄夜烤场 ┊更便宜 更安静 │
21:00    │   ●━━━━┐  └────────────────────────┘
```

## 四、顺带修：切换活动按钮按内容自适应（本批一并）

现状（Image 14）：备选按钮固定截 ~6 字，"老王烧烤摊…""夜烤场·…"被硬切、没显示完全。
改：**内容自适应宽度**——按钮宽度贴合名字（`max-width` + `truncate`），短名全显，**只在真的超长时才截**并 `title` 出全名。`max-width` 取一个合理上限（如 8–10 字宽），避免超长名撑爆整行。②具名备选和③定向微调都按此。
