# 增长路径

> 0 → 100 → 1000 → 10K+ 用户的三阶段增长策略。

---

## 一、阶段 1：0 → 100 用户（朋友圈 + 小红书）

### 1.1 触发条件

Demo 完成 + α 上线（接入高德 API）。功能稳定可用。

### 1.2 主战略：分享文案 + 小红书种草

晌午局的产品独特性恰好就是「输出转发文案」——用户接受方案后，**Agent 已经为他写好了一段可分享的文字**：

```text
example：
「今天下午带娃去童趣沙池公园（开车 12 分钟），
玩 2 小时刚好赶上 5 点的鲸落健康简餐
（人均 78，包间能装下我们一家），晚上 7 点前到家。」
```

这种文案的传播性远高于「我去了 XX 店」——**它是规划本身**，朋友看完也想用。

### 1.3 具体策略

```text
| 渠道           | 内容形式                           | 投入                |
|---------------|----------------------------------|--------------------|
| 朋友圈（创始人） | 直接发 demo 视频 + 一句话方案      | 0 元                |
| 小红书种草     | 5-10 篇「XX 场景 1 句话出方案」笔记 | 自创内容 + 0 元投放 |
| 微信社群       | 投相关本地生活群、亲子群           | 0 元                |
| 媒体投稿      | 给科技 / AI 自媒体投 Demo 视频     | 0 元                |
```

**KPI**：

```text
- 日新增用户 5+ / 天
- 7 天留存 ≥ 30%
- 自然分享率 ≥ 5%（用户主动转发分享文案的比例）
```

### 1.4 失败信号

```text
- 7 天留存 < 15%        → 不是真实需求，回炉重做产品
- 分享率 < 2%           → 文案质量不够好 / 用户不愿展示「AI 帮我决策」
- 用户都集中在亲子      → 太垂直，扩 persona
```

## 二、阶段 2：100 → 1000 用户（异业合作）

### 2.1 触发条件

PMF 信号验证（留存 ≥ 30%、confirm 率 ≥ 30%、分享率 ≥ 5%）。

### 2.2 主战略：与 3 类机构合作分发

#### 2.2.1 亲子机构 / 早教中心

```text
价值交换：
  我们：免费给机构成员发 6 个月会员卡 + 推荐机构加入「商家联盟」
  机构：在群发活动 / 课程提醒里加我们的小程序入口
  
合作目标：5 家机构 × 200 家庭 = 1000 个潜在新增
```

#### 2.2.2 企业团建 / 公司 HR

```text
价值交换：
  我们：免费给企业开「团建规划」入口（自动整合公司预算 + 团队人数 + 偏好）
  企业：在年度团建 / 季度 outing 里使用并 case study
  
合作目标：10 家中型企业 × 50 员工 = 500 潜在新增（高净值）
```

#### 2.2.3 短租平台 / OTA

```text
价值交换：
  我们：在用户预订短租房后弹出「该地区周末规划」推荐
  平台：增加用户停留时间 + 减少行程焦虑
  
合作目标：1 家平台 × 1000 月入住 = 100-200 新增
```

### 2.3 KPI

```text
- 月新增用户 200-500 / 月
- 异业合作渠道占比 ≥ 50%
- 头部 5 商圈商家直签 ≥ 20 家
```

### 2.4 关键防御

```text
- 异业合作避免独家：每个垂类合作 2-3 家，避免被任一家绑架
- 数据隔离：合作方不能拿到我们的用户行为数据
- 体验一致性：合作方入口与主入口体验一致，避免「合作方专属 bug」
```

## 三、阶段 3：1000+ 用户（投流 + SEO + 商家联盟）

### 3.1 触发条件

阶段 2 验证：异业合作可复制 + ARPU 达 ¥2 + 单位经济正向。

### 3.2 主战略

#### 3.2.1 探索投流

```text
| 渠道           | 受众                       | 预算建议（月）   |
|---------------|---------------------------|----------------|
| 小红书信息流   | 25-40 已婚女性、亲子        | ¥10k-30k       |
| 微信视频号     | 35-55 中产家庭              | ¥5k-20k        |
| 抖音           | 不优先（受众太广，CPC 偏高）| 暂缓            |
```

> 投流预算根据 LTV / CAC 动态调整，CAC ≤ ¥10 时大量投，超过则收缩。

#### 3.2.2 SEO（PGC 内容）

```text
- 「XX 商圈周末带娃去哪」类内容：SEO 长尾词
- 「商务接待选什么餐厅」类决策长文：高净值客流
- 「老人陪伴一日游路线」类系列内容：差异化用户群
```

每篇文章嵌入「试试一句话规划」入口，转化率高于普通广告。

#### 3.2.3 商家联盟

签约商家自带分发能力：

```text
- 商家在自己门店的 WiFi 登录页 / 桌面二维码引流到晌午局
- 商家给到券后引导「下次让 AI 帮你计划全场景」
- 商家联盟内的商家给晌午局「优先推荐位」对应免抽佣
```

### 3.3 KPI

```text
- MAU 10k → 100k（12 个月内）
- ARPU ¥2-3
- LTV / CAC ≥ 3
- 头部 5 商圈商家直签 ≥ 100 家
```

## 四、增长漏斗 + 关键转化点

```text
小红书 / 朋友圈曝光
        ↓ ~5% 点击率
落地页（一句话演示）
        ↓ ~30% 完成首次对话
首次方案（关键体验点）
        ↓ ~30% confirm
首次 confirm（关键留存点）
        ↓ ~50% 7 天回访
活跃用户
        ↓ ~10% 月分享
新一轮拉新
```

### 4.1 关键转化点优化方向

```text
| 关键点         | 优化方向                                  | 工具/手段              |
|---------------|------------------------------------------|---------------------|
| 落地页 → 对话   | 简化输入框 + 8 场景按钮兜底               | A/B 测试            |
| 对话 → 方案    | 防意图解析失败 / 候选过严 / LLM 抖动        | 五级降级策略         |
| 方案 → confirm | 个性化匹配 + 包含 reasoning（说明书）      | persona prior + memory |
| confirm → 留存 | 邀请加入家庭群 / 开通日历同步              | MVP 阶段功能         |
| 留存 → 分享    | 文案质量 + 分享激励（推荐有礼）            | LLM 文案优化        |
```

## 五、增长里程碑

```text
| 时间           | 用户量级         | 主要拉新渠道                       |
|----------------|----------------|-----------------------------------|
| 0 ~ 1 月（α）  | 0 → 100        | 朋友圈 + 小红书自然                  |
| 1 ~ 3 月（β）  | 100 → 1000     | 异业合作（亲子 / HR / 短租）        |
| 3 ~ 6 月（v1）| 1000 → 10k     | 投流 + SEO + 商家联盟              |
| 6 ~ 12 月      | 10k → 100k     | 大流量复制 + 美团生态合作           |
```

## 六、风险与降级

```text
| 风险                          | 降级策略                             |
|------------------------------|------------------------------------|
| 小红书种草无声量              | 切换微信视频号 / 知乎专栏             |
| 异业合作签约困难              | 降低门槛（免费 6 月会员 → 免费永久会员）|
| 投流 CAC 居高不下             | 暂停投流回到内容运营 + 自然增长       |
| LLM 成本不降                 | 自研轻量意图解析模型                  |
| 数据合规审查                  | 上线前过等保二级（β 阶段开始）        |
```

---

## 七、为什么这个路径靠谱

```text
1. 阶段 1 不烧钱：用产品独特性（分享文案）做自然增长
2. 阶段 2 借渠道：异业合作高效低成本，数据可监控
3. 阶段 3 靠工程：投流 / SEO / 商家联盟都是已验证的玩法
4. 不依赖单一渠道：每个阶段 ≥ 3 个并行渠道，单点失败可切换
5. 单位经济在阶段 2 就转正：详见 [`04-商业模式.md`](./04-商业模式.md)
```
