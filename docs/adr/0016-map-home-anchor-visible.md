# ADR-0016: 地图家锚点可见——推翻「抽象=不可见」的推论,保留「家是锚点」的域概念

- **日期**: 2026-07-10
- **状态**: Accepted
- **范围**: 前端 MapOverlay(纯渲染层;后端零改动)
- **承接**: 无直接前序 ADR;推翻的旧决策记录在 `frontend/components/MapOverlay.tsx` 头注释(「home 节点永远不画——它是抽象起终点」)。

## 背景

地图现状只画行程中间站(编号 marker + 相邻站两两真实 Driving 连线,失败退直连 Polyline),`target_kind === "home"` 的首尾节点被有意跳过。旧决策的理由:家是「抽象起终点」,不是安排给用户的活动。

用户在路演打磨中提出:**地图上看不到家,行程没有闭环感**——方案的叙事是「从家出发→玩一圈→回家」,时间轴上有出发/返程通勤壳,地图却只讲了中间一段。两个视图对同一方案讲的故事不一致。

技术前提已核实:后端 `assemble_from_blueprint` 组装时**本来就给 home 首尾节点注入了 `lat/lng`**(取自 `user_profile.home_location`,见 `assemble_blueprint.py:394/513`),数据一直在桥上,只是前端不消费。

## 决策(grilling 会话逐点拍板,2026-07-10)

**域概念澄清(本 ADR 的核心)**:旧决策捆绑了两件事——「家是锚点不是站」(域概念)和「所以不画」(渲染推论)。本 ADR **保留前者、推翻后者**:抽象锚点也可以可见,身份区分由**形态**负责,闭环感由**连线**负责。

### 1. 身份:锚点,不是「第 0 站」

家用独立形态 marker——**墨色底 + 白色 Lucide `Home` 图标**的 pin,**不占编号**;编号站维持黄底数字。理由:给家编号会稀释「N 站方案」的语义(评委数 marker 会数错);且与时间轴语言一致(家在时间轴上是通勤壳,不是节点卡)。

### 2. 闭环连线:通勤壳用虚线,站间用实线

家→首站、末站→家两段画**虚线**(同色、稍细);站间维持实线。两段通勤壳照样走真实 Driving 路线、失败退直连——与站间同一套降级逻辑,不另造。理由:时间轴上这两段本来就是「出发/返程」通勤壳,地图与时间轴讲同一套语言;虚线在地图语言里天然读作「往返/接驳」;首尾腿常是全图最长线,虚线让通勤不抢行程的戏。

### 3. 编舞:家最先,返程线最后

家锚点 pin 在地图初始化**即刻就位**(「你从这儿出发」的原点,不属于方案揭晓的悬念);编号站照旧随时间轴进度 stagger 冒出;去程虚线随 1 号站一起画出;**返程虚线在最后一站落定后收尾一笔画回家**——闭环成为编舞的收束动作。附带好处:家先出现给 Driving 路线请求提前量。

### 4. 细节四则

- **pin 不写字**:图标自明;点击弹 InfoWindow 显示画像家名(路演时=「望京数字创意园B座2号楼」),与站点 InfoWindow 同交互语言。
- **fitView 包含家**:闭环必须一屏看全。
- **降级**:home 节点缺坐标 → 静默回退到现状(不画家、不画虚线腿,`console.warn` 留诊断),多用户画像缺数据时地图不崩。
- **配色层级**:「锚点中性、行程主角」——与信任带检索芯片(中性墨色、琥珀留给自愈拍)同一套视觉哲学。

## 备选与否决

- **家=第 0 站(编号 marker)**:稀释站数语义,时间轴/地图概念漂移 → 否。
- **通勤壳与站间同样式实线**:最长的通勤腿抢行程主角戏 → 否。
- **只画去程不画返程**:闭环正是诉求,砍返程等于没闭环 → 否。
- **全元素同时出现**:浪费现有 stagger 编舞的叙事感 → 否。

## 落地

- 纯前端:`MapOverlay.tsx` 的 marker 解析(取消 home 跳过、独立形态)、连线分段样式、编舞时序、fitView、InfoWindow、降级分支。
- **时序约束**:MapOverlay 正被望京数据迁移任务占用(改 FALLBACK_CENTER),本 ADR 实现排在其后,避免同文件冲突。
- 落地时同步改写 MapOverlay 头注释的旧决策文案,指向本 ADR。

## 词汇表增量

- **家锚点(home anchor)**:行程的起终点抽象,可见但不占编号、不算站;数据形态是 `target_kind="home"` 的首尾节点(带坐标)。
- **通勤壳(commute shell)**:家↔首末站的出发/返程段;时间轴上是壳不是节点卡,地图上是虚线不是实线。
