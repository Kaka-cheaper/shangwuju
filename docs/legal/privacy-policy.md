# 隐私政策（占位草案）

> 本文件为「晌午局」上线前的法务占位草案，**未经执业律师审核，不能直接作为正式协议使用**。
> 上线前请由律师按 GDPR / 个人信息保护法 / 网络安全法等具体法规审核。

更新日期：2026-05-22  
版本：v0.1（占位）

---

## 一、本政策适用范围

本政策说明「晌午局」（以下简称「本服务」）如何收集、使用、存储、保护您的个人信息。

## 二、我们收集哪些信息

### 2.1 您主动提供的信息
- 第三方账号授权信息（OpenID / UnionID / 昵称 / 头像）
- 您输入的出行需求自然语言文本
- 您主动选择 / 反馈的出行偏好

### 2.2 自动收集的信息
- 设备类型与浏览器信息（用于适配 UI）
- 访问日志（IP / 时间戳 / 路径，用于安全风控）
- 大致地理位置（仅在您授权时获取，用于附近 POI 搜索）

### 2.3 我们不会收集
- 精确实时位置（除非您主动开启）
- 通讯录 / 短信 / 相册等敏感设备数据
- 银行卡 / 支付密码（支付走第三方支付通道）

## 三、我们如何使用信息

| 用途                            | 说明                                                       |
|---------------------------------|------------------------------------------------------------|
| 行程规划                        | 把您的输入传给 LLM 服务做意图抽取与方案生成                |
| 附近 POI 搜索                   | 把您的位置（经纬度 + 半径）传给地图服务做附近搜索          |
| 用户偏好积累                    | 您接受 / 拒绝的方案 tag 写入个人偏好库，用于下次推荐排序   |
| 安全风控                        | 异常访问 / 高频请求识别                                    |

## 四、我们如何共享信息

我们仅在以下情况共享信息：
- **第三方 LLM 服务**：您输入的出行需求文本会传给 DeepSeek / 通义千问等以生成方案，传输前去除可识别个人身份信息
- **第三方地图服务**：您的大致位置会传给高德等以查询附近 POI
- **法律强制要求**：依据司法机关合法请求

我们承诺不会把您的信息用于本服务范围外的商业用途。

## 五、信息存储与保护

- 会话数据默认保留 24 小时（详见 `SESSION_STORE` 配置）；您可随时通过「重置」按钮清除
- 偏好数据持久存储；您可通过「清空偏好」一键删除
- 所有数据传输使用 HTTPS / WSS 加密
- 后端密钥（LLM / 地图 API key）通过 env 注入，不入库不入日志

## 六、您的权利

依据《个人信息保护法》，您拥有：
- **知情权**：可随时查询我们持有的您的信息（通过「我的偏好」面板）
- **删除权**：可随时清空个人偏好与会话历史
- **撤回授权**：可在第三方账号管理面板撤回登录授权
- **投诉权**：通过下方邮箱提交投诉

## 七、儿童隐私

本服务不主动收集 14 岁以下儿童的信息。如果您是监护人发现孩子未经允许使用本服务，请联系我们删除相关数据。

## 八、Cookie 与本地存储

本服务使用以下 Cookie / 本地存储：
- 必需 Cookie：会话标识 / 用户偏好（不可关闭）
- 分析 Cookie：访问统计（您可在浏览器设置关闭）
- 我们不使用任何第三方追踪 Cookie

## 九、政策变更

本政策更新时通过站内通知告知您；重大变更（涉及收集范围扩大）会要求您重新授权。
